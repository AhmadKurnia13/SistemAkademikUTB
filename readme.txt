DESKRIPSI PROYEK
Sistem Informasi Akademik UTB adalah aplikasi perangkat lunak 
berbasis Java Desktop yang dirancang untuk mempermudah pengelolaan 
data akademik. Dibangun dengan JDesktopPane dan JInternalFrame, 
aplikasi ini mengusung desain Premium Dark Mode yang elegan dan
profesional. Aplikasi ini terintegrasi penuh dengan Database MySQL.

----------------------------------------------------------------------

FITUR UTAMA APLIKASI
1. Secure Login System
   Autentikasi pengguna dengan validasi langsung dari database.
2. Premium Dark Mode UI
   Antarmuka pengguna modern dan elegan (Pure Java Swing).
3. Master Data Management (CRUD)
   Pengelolaan Data User, Mahasiswa, Dosen, dan Mata Kuliah.
4. Modul Transaksi Akademik
   - KRS: Pendaftaran mata kuliah per semester.
   - Input Nilai: Form penginputan nilai huruf terstandarisasi.
5. Dynamic Data Rendering
   Penampilan data real-time pada tabel (JTable) tanpa border kaku.

CARA MENJALANKAN APLIKASI

[ Tahap 1: Persiapan Database (XAMPP) ]
1. Buka XAMPP Control Panel.
2. Start modul Apache dan MySQL.
3. Buka browser dan akses http://localhost/phpmyadmin.
4. Buat database baru dengan nama: db_uts_pbo_ahmad.
5. Lakukan Import struktur tabel jika tersedia.

[ Tahap 2: Konfigurasi IDE (NetBeans) ]
1. Buka Apache NetBeans IDE.
2. Open Project "SistemAkademikUTB".
3. Pastikan driver "mysql-connector-j-9.7.0.jar" sudah ditambahkan 
   pada bagian Libraries proyek.

[ Tahap 3: Eksekusi Program ]
1. Cari file utama penggerak aplikasi: 
   src/Akademik/SistemAkademikUTB.java
2. Klik kanan pada file tersebut dan pilih "Run File".
3. Login menggunakan akun Anda untuk mengakses Dashboard Menu Utama.
